Attributes

Here's a right-aligned paragraph, which looks like this <p align = "right">

Most HTML elements have the following attriubtes: id (which can be used to uniquely identify any element); title (which functions in a similar way, but can also serve as as a tooltip or something); class (which associates an element with a particular class within a Cascading Style Sheet); and style (which allow you to specify specific CSS features for that element).

Other common attributes are: dir (which can have values "rtl" or "ltr" to define which way text is supposed to be read); align (horizontal); valign (vertical); bgcolor (when given a numeric, hexidecimal, or RGB colour, places it in the background of a particular element); background> (when given a URL, places the image from that URL as the background of the element); width; height. And some others...

The tag <tt> is used to present text in monospaced font. Also, <strike>, <sup>, and <sub>, which should be obvious. Some more are <big> big, and <small> small.

<div> and <span> can be used to divide a page into smaller sections (span only encompasses inline elements, whereas div can be larger).
